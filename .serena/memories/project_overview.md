# 开口鸭项目概览

## 项目简介
开口鸭（Open Duck）是一个基于 **Taro 4 + React 18 + TypeScript** 的跨平台英语学习应用，专注于口语练习和语言学习。

## 核心功能
- **语音对话**: AI 智能对话练习
- **翻译功能**: 中英文实时翻译
- **单词学习**: 词汇记忆和练习
- **话题练习**: 场景化对话训练
- **拍照短文**: 图片生成英语故事
- **学习进度**: 个人学习数据统计
- **会员系统**: 高级功能订阅

## 支持平台
- H5 (Web)
- 微信小程序
- 支付宝小程序
- 百度小程序
- 字节跳动小程序
- QQ 小程序
- 京东小程序
- 快应用
- React Native
